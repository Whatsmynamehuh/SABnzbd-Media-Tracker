# Public Assets

This folder contains static assets served by the app.

## Logo/Favicon

Add your logo here as `favicon.png` or `favicon.ico`:

- **Recommended format**: PNG with transparency
- **Recommended size**: 512x512px (will auto-scale for browser tab)
- **File name**: `favicon.png` or `favicon.ico`

The logo will appear in:
- Browser tab icon
- Bookmarks
- Mobile home screen (when added)
